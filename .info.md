# 클립보드 공유 프로젝트

2대 이상의 PC 간 텍스트를 실시간으로 공유할 수 있는 독립 웹 애플리케이션

## 빠른 링크
- [README.md](README.md) - 프로젝트 개요 및 기능 소개
- [SETUP.md](SETUP.md) - 상세한 설치 및 설정 가이드

## 프로젝트 파일
```
clipboard-share/
├── index.html              # 메인 페이지
├── css/
│   └── styles.css          # 스타일시트
├── js/
│   ├── config.js           # Firebase 설정 (수정 필요)
│   └── app.js              # 메인 로직
├── firebase.json           # Firebase 호스팅 설정
├── database.rules.json     # Database 보안 규칙
├── README.md               # 프로젝트 문서
└── SETUP.md                # 설치 가이드
```

## 시작하기
1. [SETUP.md](SETUP.md)를 따라 Firebase 프로젝트를 설정하세요
2. `js/config.js`에 Firebase 구성 정보를 입력하세요
3. 웹 서버를 통해 `index.html`을 실행하세요

독립 프로젝트로 자유롭게 수정 및 배포 가능합니다.
